# ig
Simple photo sharing app inspired by Instagram. Take photos, post to the web using the Firebase SDK, photos are stored in Firebase Storage. References to the photos are stored as JSON. Using the Firebase Database REST API, makes HTTP requests to add likes and comments. Interface is designed with a Tab Bar Controller and a UICollectionViewController.
