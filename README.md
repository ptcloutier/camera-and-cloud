# camera-and-cloud
Posts photos to the cloud and allows users to see all photos posted. Users can leave comments and likes. 
Using the Firebase SDK, photos are stored in Firebase Storage. References to the photos are stored as JSON. Using the Firebase Database REST API, makes HTTP requests to add likes and comments. Interface is designed with a Tab Bar Controller and a UICollectionViewController.
